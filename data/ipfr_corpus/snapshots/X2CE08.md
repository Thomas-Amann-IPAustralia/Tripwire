Collection description
This collection contains the Australian Respiratory Surveillance Reports (ARSR), along with the Technical supplement – Australian Respiratory Surveillance Report.
We compile each report from several data sources used to monitor the:
- distribution of acute respiratory illness activity in the community
- severity of infections
- populations which might be at-risk for severe infections
- impact of acute respiratory illness activity on the community and health system in Australia.
The reports are published:
- each fortnight from April to October
- each month from November to March.
For historical information on influenza surveillance in Australia, see the Australian Influenza Surveillance Reports.
For historical information on coronavirus disease 2019 (COVID-19) surveillance in Australia, see the COVID-19 Australia: Epidemiology reports.
Read also the Australian National Surveillance Plan for COVID-19, Influenza, and RSV.

Technical supplement
You can view the current Technical supplement – Australian Respiratory Surveillance Report here. This supplement to the ARSR series describes the technical background to the surveillance data reported nationally for:
- COVID-19
- influenza
- respiratory syncytial virus (RSV)
- other acute respiratory pathogens.
The supplement includes data considerations to help interpret the ARSR.
